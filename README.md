# ZeroToHeroColabCollection
I have been following and completing the Zero to Hero series by Andrej Karpathy. This repo is strictly for me to organize and dump the notebooks I made while watching him, and the notebooks I create the next day on a blank notebook to improve my recall of the content. Disclaimer: Some small issues might exist in my recall notebooks for obv reasons.

There are two folders. The first is guided reference builds. As the name suggests, I made them while watching Karpathy. The second one is unassisted rebuilds, which as the name suggests is me rewriting the code, but from on a blank notebook from memory. The 2nd type is probably prone to errors and shortcuts.
